## OpenMCP

一款用于 MCP 服务端调试的一体化 vscode 插件。

- 包含原版 Inpsector 的所有功能
- 包含一个简易的用于进行测试的大模型对话 & 执行窗口
- 支持多种大模型


## Dev

- `app`: 前端 UI 的定义
- `test`: 测试 `app` 的部分，包含一个简易的转发层
- `src`: vscode 插件端定义
